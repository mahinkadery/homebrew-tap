# FlowShelf Homebrew Tap

Official [Homebrew](https://brew.sh) tap for [FlowShelf](https://flowshelf.app) — the free, notarized macOS menu-bar shelf, clipboard manager and screenshot studio.

## Install

```sh
brew install --cask mahinkadery/tap/flowshelf
```

## Upgrade

```sh
brew upgrade --cask flowshelf
```

FlowShelf also updates itself in-app via Sparkle. Both paths ship the same signed, Apple-notarized build from [GitHub Releases](https://github.com/mahinkadery/FlowShelf/releases).
